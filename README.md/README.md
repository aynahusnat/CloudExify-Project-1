## Project Introduction

This project is a personal portfolio website designed and developed for Ayna Husnat. The purpose of the website is to present a professional online identity, showcase web development skills, highlight selected projects, and provide clear contact information for future opportunities.

The portfolio is created as a static multi-page website using HTML, CSS, and vanilla JavaScript. It is lightweight, responsive, and suitable for deployment on GitHub and Vercel.

## Project Objective

The main objective of this project is to create a clean and professional portfolio that represents Ayna Husnat as a web developer. The website focuses on readability, responsive design, useful interactivity, and a polished visual presentation.

The project also demonstrates semantic HTML structure, CSS styling, responsive layout design, DOM manipulation, JavaScript events, and frontend interactivity.

## Project Description

The portfolio contains four main pages. The Home page introduces the developer with a professional hero section, a typewriter animation, a downloadable CV button, and visual project artwork. The About page explains the developer profile and includes animated skill bars. The Projects page displays selected project cards with filtering functionality. The Contact page presents contact details in a clear right-side section.

## Features Implemented

- Multi-page portfolio website
- Responsive navigation menu
- Mobile hamburger menu
- Light and dark theme switcher
- Theme preference saved in `localStorage`
- Typewriter animation on the Home page
- Animated skill progress bars
- Project filtering by category
- Downloadable CV button
- Contact information section
- Local SVG visual assets
- Responsive layout for desktop and mobile screens

## Technologies Used

- HTML5 for page structure
- CSS3 for styling, layout, responsiveness, and theme design
- Vanilla JavaScript for interactivity
- SVG images for lightweight visual graphics

## Design Summary

The website follows a clean editorial style with Times New Roman typography. The current color system uses a full-screen skin-toned interface, softly darker content panels, warm brown text, and a separate blue dark theme with strong contrast.

## JavaScript Functionality

JavaScript is used to control the mobile menu, theme switcher, typewriter animation, project filtering, reveal animations, and skill bar animation. These features make the portfolio more interactive while keeping the code simple and readable.
